Mandelbrot Movie Maker

Erkunden Sie die Mandelbrot-Menge. Mit der Maus kann ein rechteckiger Bereich zum Vergrössern ausgewählt werden. Die Auswahl kann mit den Pfeiltasten feinjustiert werden. Mit der Return-Taste wird vergrössert. Das Programm benutzt die 64-Bit IEEE 754 Gleitkommadarstellung für Berechnungen, daher gibt es Limitationen für den Zoombereich. Jeder Vergrösserungsschritt wird in der Vorschau dargestellt. Die jeweiligen Vorschauen sind anwählbar. Wenn die Entfernen-Taste gedrückt wird, wird die Vorschau sowie alle rechts von ihr befindlichen gelöscht.

Filmerzeugung

Aus den einzelnen Vergrösserungsschritten kann ein Film erzeugt werden. Wenn diese Aktion ausgewählt wird, muss die Ausgabedatei sowie die Bilderwiederholrate angegeben werden. Die Anzahl von Bildern pro Vergrösserung wird zu Beginn des Programms festgelegt, ebenso wie der Farbverlauf.

Einzelne Bilder speichern

Die aktuell angezeigte Vorschau kann als PNG Bild gespeichert werden. Wählen Sie die entsprechende Aktion im Datei-Menü. Das so gespeicherte Bild enthält Metadaten, welche den aktuellen Vergrösserungsschritt beschreibt, und kann beim Start des Programms auch wieder geladen werden.
